# Security Policy

## Supported versions

The current main branch is supported for security fixes.

## Reporting a vulnerability

If you discover a security issue, report it privately to the maintainer through the appropriate private channel for your repository hosting platform.

Please include:

- A description of the issue.
- The impact you believe it may have.
- Reproduction steps, if available.
- Any suggested mitigation.

Do not publish the vulnerability publicly until it has been reviewed and addressed.
